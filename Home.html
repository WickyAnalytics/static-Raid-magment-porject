Hi everyone !
I made a tool using Google Sheets to help organising weekly raids with my static.
I share it so anyone can make a copy and use it too if you think you need it.
Please give me some feedbacks ! (Crowicky#0000 on discord)
I will try to update it to fix some issues in the future but at the moment it's quite stable as we use it my actual static.

This sheets contains 5 useful tabs, the others are used to compute data and are hidden by default.
I will explain each tabs below:
8
9
10
Schedule
11
12
This tab contains a schedule for all unique players listed in the Characters tab.
Anyone can set anyone unavailable/Available by putting an "X","Y" the desired cell. By doing so the cell should automaticaly turn Red/Green.
It is useful for two major things:
The first one is to quickly see the shifts were 4 or 8 people are available and plan a raid at this time.
The second one is to see quickly in the Planner tab if a raid is planned with someone that is not available.
All players should check and update if necessary on Monday or Tuesday to be sure all raids for the Lost Ark week can be done without issues.

15
16
Planner
17
18
This tab contains all planned raid compositions and time.
You can also check if the raid is done, all checkbox will be automaticaly unchecked at weekly reset (Wednesday 12h).
In each raid wrapper, you can select the raid, the characters, day, start and end time.
You can't select the same character twice in the same type of raid (ex: Akkan NM) in the dropdowns but it can be done manualy.
If you select a character from a player that is not available at the start shift or end shift his name will be highlighted in red in to alert you from his unvailability. It works only if the Schedule tab is properly filled.
You can quickly see who's doing the raid and what are the synergies per party thanks to the conditional formating.
Do not ever copy paste characters using CTRL+C and CTRL+V it will mess up the dropdowns, use CTRL+SHIFT+V to paste the value only instead.
If you mess up something keep in mind you can always CTRL+W to cancel the last action made on the document.
